Movie project

Context

During this project, I had to apply all my gained knowledge of HTML and CSS into creating a fake movie webpage. We needed to work with a partner in this SEP Movie Project, and so we decided to make a narrative movie, titled, “Time Bending Detective".

Content

To start our project, we needed to make a plan. We plan our plot and what we should include in the movie for the webpage. This is for HTML part:

  • headings
  • paragraph
  • divs and spans
  • imported Google fonts
  • inputs and submit button
This is for the CSS part:
  • element and class and id selectors
  • border and padding and margin properties
  • two local images

Challenges

My partner and I tried our best in this project. A difficulty that we had was brainstorming a story. I was in charge of making the story but it was hard to think of an idea for it. So I asked my partner what genre for the movie we should choose, and we ended up choosing a narrative mystery. With that idea I started to plan on the storyline of it. It was now much easier to create a storyline.

Takeaways

I learned that if you're facing a challenge you should always try to ask your partner for help, it’s easier with two brains thinking than one. Splitting the project doesn’t mean you have to work alone on that part of the project, it’s just easier to see who’s working on what to make it organized.

Next Step

I will make the webpage look better if I have more time. I already had some ideas of how to make it better but we did not have enough time. For the future, I’m hoping to make the project look even better with visual designs, making it more visually appealing.
